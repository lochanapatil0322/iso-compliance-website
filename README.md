# ISO Compliance & Audit Knowledge Hub

A cybersecurity internship website that explains ISO 27001, ISO/IEC 27002, ISO 27001 audit checklist practices, and ISO 42001:2023 AI management system audit controls.

## Pages

- `index.html` - Home page and project introduction
- `iso27001.html` - ISO 27001 implementation and auditing overview
- `iso27002.html` - ISO/IEC 27002 security controls overview
- `iso27001-checklist.html` - Interactive ISO 27001 audit checklist
- `iso42001.html` - ISO 42001:2023 AI governance overview
- `comparison.html` - Comparison table for all documents

## Features

- Professional static website suitable for GitHub Pages
- Separate pages for each ISO topic
- Interactive checklist progress counter
- Search box for checklist items
- Responsive layout for desktop and mobile
- Original educational summaries

## Technologies Used

- HTML
- CSS
- JavaScript
- SVG visual asset

## How To Run

Open `index.html` in a browser.

## GitHub Pages Deployment

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to repository Settings.
4. Open Pages.
5. Choose Deploy from branch.
6. Select `main` and `/root`.
7. Save and wait for the GitHub Pages link.
